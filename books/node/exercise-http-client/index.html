In this exercise, we will show how you can upload a form, we will show how you can create a form and write server code to handle the submission of that form.

# Upload a form

For this version we'll create two different routes:

  • /form, when requested with a GET verb we'll render out HTML content that consists of an HTML form and some input elements
  • /submit, when requested with a POST verb we'll receive posted data

Let's build our server application. Start with creating a file a server.js and give it the following content:
