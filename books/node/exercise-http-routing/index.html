## Exercise - handling static files

It's usually to place all static assets in a dedicated folder in your program. An even better solution is to put those assets in a Storage in the Cloud and use CDN, but that's outside the scope of this module.

Let's start by creating our application file server.js.

Next create a folder static. Inside of that folder creates a file and gives it the following content:

To serve requests asking for static files, we will have to look for request URLs containing the pattern /static/. We will also need to ensure that we are checking so that the request URL ends in a permitted file extension. Add the following code to server.js:

const http = require('http');
const path = require('path');
const fs = require('fs');

const server = http.createServer((req, res) => {
  // if part of url is static then treat the same
  if (/\/static\//.test(req.url) && /\.{1}txt|jpeg|css{1}$/.test(req.url)) {
    const read = fs.createReadStream(path.join(__dirname , req.url));
    read.pipe(res);
  } else {
    res.end('ok')
  }
})

server.listen(8000, () => {
  console.log('Server running on 8000')
});

# Exercise - matching different routes

We've learned how to handle static routes. Let's work on routes that don't ask for specific files. Let's learn to handle routes looking like this:

  • /products, this is a route should return a list of products. Valid versions of this route are /products and /products
  • /products/114, this should return a single product with id 114.

Let's create a new file server-routes.js and give it the following content:

const http = require('http');

const server = http.createServer((req, res) => {
  const products = [{
    id: 114,
    name: 'a learn module'
  }]

  if (/\/products\/\d+$/.test(req.url)) {
     const [productId] = req.url.match(/\d+$/)

    res.writeHead(200, { 'Content-Type': 'application/json' })
    res.end(JSON.stringify(products.find(p => p.id == productId)))
  } else if (/\/products\/?$/.test(req.url)) {
    res.writeHead(200, { 'Content-Type': 'application/json' })
    res.end(JSON.stringify(products))
  } else {
    res.writeHead(404, { 'Content-Type': 'text/plain' })
    res.end('Not found')
  }
});

server.listen(8000, () => {
  console.log('Server running on port 8000')
});
