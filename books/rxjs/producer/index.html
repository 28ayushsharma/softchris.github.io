# Producer

Follow me on Twitter, happy to take your suggestions on topics or improvements /Chris

A producer have the task of producing the values emitted by an Observable

class Producer {
  constructor(){
    this.i = 0;
  }

  nextValue(){
    return this.i++;
  }
}
